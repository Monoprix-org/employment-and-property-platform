# Employment and Property Platform

## Project Overview
A full-stack platform for job posting, job applications, and property advertisements.

## Tech Stack
- Node.js
- Express.js
- MySQL
- React

## Backend Structure
- db: database connection/config
- middleware: authentication and other middleware
- routes: API routes
- server.js: main backend entry

## How to Run
1. Install dependencies:
   npm install

2. Create a `.env` file based on `.env.example`

3. Start the server:
   node server.js

## Main Features
- User registration and login
- JWT authentication
- Job posting
- Job applications
- Property advertisement module
